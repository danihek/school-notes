# Benzen

- działa drażniąco
- może powodować raka
- jest substancją mutagenną i rakotwórczą
- jet łatwopalny

Wewnątrz cząsteczki benzenu wystąpują wiązania zdelokalizowane

## Właściwości chemiczne

## Reakcja SUBSTYTUCJI

$C_6H_6 + Br_2 → C_6H_5Br + HBr$

Benzen → Bromobenzen

## Reakcja Nitrowania

$C_6H_6 + HNO_3 → C_6H_5NO_2 + H_2O$

Benzen → Nitrobenzen
